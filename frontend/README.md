# CS Learning Map — Frontend

Next.js 14 (App Router) learner and admin UI for the adaptive CS learning system: knowledge map (React Flow), practice, progress, and optional AI tutor panel.

**Setup, Docker, API URLs, and grading:** see the [repository root README](../README.md).

```bash
npm install
cp .env.local.example .env.local
npm run dev
```

Deploy like any standard Next.js app (e.g. Node host or container) with `NEXT_PUBLIC_API_URL` pointing at your FastAPI base URL.
